This project depends on https://github.com/pingpongboss/StandOut for UI elements, and you will have to clone and build the library before the project will build. 

If you're using ant, modify the build.xml to point to the jar location, or manually copy the jars into the project's library path.

Installation
------------

Note: Requires Android 4.1 or newer

1. Allow your device to install apps from outside the Play Store:
	Enable Setting->Security->Unknown Sources
2. Copy APK to device
	Connect device via USB as a media device
	Copy APK anywhere to device memory
3. Install APK
	Navigate to the location of the APK by whatever file explorer you have (If you don't have one, ASTRO is good. It is in the Play Store)
	Click the APK. Installation should start automatically
